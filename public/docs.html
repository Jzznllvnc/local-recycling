Everything you need to know about finding and using recycling centers with our platform.

Introduction

Welcome to the documentation for ScrapNear. This guide provides an overview of the application's features and how to use them effectively. Our mission is to make recycling accessible to everyone by providing an easy-to-use tool for finding local recycling centers.

How the Map Works

The core of ScrapNear is its interactive map, powered by Leaflet.js and OpenStreetMap. When you search for a location or use your device's location, our application sends a request to the Overpass API. This API queries the vast OpenStreetMap database to find recycling facilities within a 5km radius of your search point.

We then display the 5 closest results that have a registered name, ensuring you get the most relevant and useful information.

Tip: For best results, allow location access in your browser to quickly find recycling centers near you.

Data Accuracy

All location data is sourced directly from OpenStreetMap, a community-driven project similar to Wikipedia. This means the accuracy of the data depends on contributions from people all over the world.

If you notice an error or a missing location, you can contribute to OpenStreetMap to help improve the data for everyone.
